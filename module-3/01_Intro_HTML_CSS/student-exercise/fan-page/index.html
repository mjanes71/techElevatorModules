What is a Goldendoodle?

A Goldendoodle is a mix between a Golden Retriever and a Poodle. There are many variations of doodles. Check out the table below to see how a Goldendoodle can come to be!

Parent 1 Parent 2 Offspring
Golden Retriever Standard Poodle F1 Standard Goldendoodle
Golden Retriever Miniature Poodle F1 Minature Goldendoodle
Golden Retriever Teacup Poodle F1 Teacup Goldendoodle
Goldendoodle Standard Poodle F1B Goldendoodle
Goldendoodle Goldendoodle F2 Double Doodle
